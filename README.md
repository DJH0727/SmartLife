# 智能家居应用

这是一个智能家居应用程序项目，旨在为用户提供便捷的家庭设备管理体验。

## 功能特点

- 提供智能家居设备控制界面
- 支持设备状态显示
- 包含AI智能控制功能
- 提供用户个人资料管理功能

## 目录结构

```
AppScope/          # 应用全局资源文件
build-profile.json5 # 构建配置文件
code-linter.json5  # 代码检查配置
entry/             # 应用主模块
hvigor/            # 构建配置目录
oh-package.json5   # 包管理配置
```

## 主要页面

- 首页 (Index.ets)
- 测试页面 (Test.ets)
- AI控制页面
- 个人资料页面

## 资源文件

包含多种图片资源和配置文件，支持不同分辨率和主题样式。

## 开发环境

基于ETS(Enhanced TypeScript)开发，适用于HarmonyOS系统。

## 测试

包含单元测试和UI测试用例，确保应用稳定性。
